1. Wprowadzenie
Regresja liniowa jest jedną z najważniejszych metod statystycznych i ekonometrycznych,
służących do analizy zależności pomiędzy zmienną zależną (objaśnianą) a jedną lub większą
liczbą zmiennych niezależnych (objaśniających). Celem jest znalezienie takiego równania
liniowego, które najlepiej opisuje tę zależność.
Ogólna postać modelu regresji liniowej to:
Prosta regresja liniowa
𝒀 = 𝜷𝟎 + 𝜷𝟏𝑿 + 𝜺
Wieloraka regresja liniowa
𝒀 = 𝜷𝟎 + 𝜷𝟏𝑿𝟏 + 𝜷𝟐𝑿𝟐 + ⋯ + 𝜷𝒑𝑿𝒑 + 𝜺
gdzie:
 𝑌 – zmienna zależna,
 𝑋𝑖 – zmienne niezależne (predyktory),
 𝛽0 – wyraz wolny (przecięcie z osią 𝑌),
 𝛽𝑖 – współczynniki regresji (wpływ 𝑋𝑖 na 𝑌),
Strona 2 z 5
 𝜀 – składnik losowy (błąd).
Przykład
Chcemy przewidzieć cenę mieszkania (Y) na podstawie powierzchni (X):
𝐶𝑒𝑛𝑎 = 50,000 + 3,000 ⋅ 𝑃𝑜𝑤𝑖𝑒𝑟𝑧𝑐ℎ𝑛𝑖𝑎 + 𝜀
Interpretacja:
 każdy dodatkowy m² zwiększa cenę mieszkania średnio o 3 000,
 50 000 to cena bazowa (przecięcie z osią Y).
2. Geneza terminu „regresja”
Termin „regresja” wprowadził Francis Galton w XIX wieku, badając zależność wzrostu
dzieci od wzrostu rodziców. Zauważył on, że dzieci bardzo wysokich rodziców są zwykle
nieco niższe, a dzieci bardzo niskich rodziców – nieco wyższe. Zjawisko to określił jako
„regresję ku przeciętności” (regression toward mediocrity). Termin ten został następnie
rozwinięty przez Karla Pearsona i innych statystyków w formalną metodę analizy
zależności między zmiennymi.
3. Metoda najmniejszych kwadratów (MNK)
Metoda najmniejszych kwadratów jest podstawową techniką estymacji parametrów
regresji liniowej. Jej celem jest takie dopasowanie prostej, aby suma kwadratów odchyleń
wartości obserwowanych od przewidywanych była minimalna. Dzięki temu otrzymujemy
najlepsze dopasowanie w sensie średniokwadratowym.

4. Założenia regresji liniowej
Aby model regresji liniowej był poprawny i dawał wiarygodne wnioski, powinny być
spełnione następujące założenia:
Strona 3 z 5
- Liniowość: zależność między zmienną zależną a niezależnymi jest liniowa.
- Normalność reszt: składniki losowe mają rozkład normalny o średniej 0.
- Homoscedastyczność: wariancja składników losowych jest stała.
- Niezależność reszt: reszty nie są skorelowane.
- Brak współliniowości: zmienne niezależne nie są silnie skorelowane między sobą.

6. Program w Pythonie
Poniższy kod w Pythonie realizuje regresję liniową między zmienną 'tip' (napiwek) a
'total_bill' (rachunek):
import seaborn as sns
import statsmodels.api as sm
import matplotlib.pyplot as plt

# Wczytanie danych
tips = sns.load_dataset("tips")

# Przygotowanie zmiennych
y = tips["tip"]
X = tips["total_bill"]
X = sm.add_constant(X) #

# Dopasowanie modelu
model = sm.OLS(y, X).fit()
print(model.summary())

# Wizualizacja
plt.scatter(tips["total_bill"], tips["tip"], alpha=0.5,
label="Dane")
plt.plot(tips["total_bill"], model.predict(X), color="red",
label="Linia regresji")
plt.xlabel("Total bill ($)")
plt.ylabel("Tip ($)")
plt.title("Regresja liniowa: napiwek vs rachunek")
plt.legend()
plt.show()

7. Omówienie wyników
Model regresji liniowej dopasowany do danych pokazuje, że wraz ze wzrostem wartości
rachunku, napiwek również średnio rośnie. Współczynnik kierunkowy β₁ można
interpretować jako przeciętną zmianę napiwku dla każdej jednostki zmiany rachunku.
Wartość R² informuje, jaka część zmienności napiwków jest wyjaśniana przez model.
